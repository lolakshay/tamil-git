# üáÆüá≥ Tamil Git Automation

A beginner-friendly **open-source Git automation project** built especially for **Tamil students** learning Git, open source, and command-line tools.

This project simplifies Git usage by converting multiple Git commands into **one simple Tamil-style command**

> üí° Learn Git without fear  
> üí° Automate repetitive work  
> üí° Build tools in your own style

| Tamil Command (File) | Original Git Command | Meaning |
| :--- | :--- | :--- |
| **`arambi`** | `git init` | **Start**: Initializes a new git repo. |
| **`copypannu`** | `git clone [url]` | **Copy**: Clones a repository from GitHub. |
| **`b-set-pannu`** | `git branch -M main` | **Set Branch**: Renames branch or sets upstream. |
| **`pullpannu`** | `git pull` | **Pull**: Gets latest changes from the cloud. |
| **`commitpannu`** | `git commit -m "msg"` | **Commit**: Saves your changes locally. |
| **`pushpannu`** | `git push` | **Push**: Uploads changes to GitHub. |
| **`deploypannu`** | *(Add + Commit + Push)* | **Deploy**: The all-in-one magic command. |

‚ö†Ô∏è Prerequisite

Make sure you have Git installed.
Step 1: Create the Folder
Go to your C: Drive (or Local Disk C).
Create a new folder named TamilGit.
Paste all the downloaded .bat files (deploypannu.bat, arambi.bat, etc.) inside this folder.

      Your path should look like: C:\TamilGit\
Step 2: Connect to Windows (The Magic Step)

We need to tell Windows where to find these commands.
Press the Windows Key on your keyboard and type: env
Select "Edit the system environment variables".
Click the button "Environment Variables..." (bottom right).
In the top box (User variables), look for a row that says Path.
Select Path and click Edit.
Click New.

    Type: C:\TamilGit
Click OK on all three windows to close them.

Step 3: Restart and Test
Close any command prompts or VS Code windows you have open
üèÅ How to start your first project?

Now that it is installed, go to a new folder where you want to write code and type:

      arambi
